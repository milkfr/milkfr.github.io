### 用户相关
```$xslt
$ sudo useradd -d /home/username -m -s /bin/bash username
-m 路径不存在会自动创建
-d 指定用户主目录路径
-s 指定默认shell
以上指定参数可以在/etc/passwd中修改，但是建议用useradd操作，修改/etc/passwd容易出现其他问题
$ sudo passwd username
$ su root
$ vim /etc/sudoers
# /etc/sudoers，加入下行增加sudo权限
username ALL=(ALL) ALL 
```
